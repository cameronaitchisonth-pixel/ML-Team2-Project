# ML Team 2 Project
Cameron Aitchison, Richard Mendez, Brandon LeMay, Dipak Sakharkar  
Machine Learning class project exploring market data.   
This repo contains all code, data handling, and reports for the project.

## Usage

Training can viewed in `Lab.ipynb`
or by finding member models in notebooks/

---

## Repository Structure

- **data/**  
  - `README.md`
  - `SPX.csv` - Data file

- **notebooks/**  
  - Individual notebooks for each team member to explore models

- **figures/**  
  - Figures from model development

- **Lab.ipynb** — Final lab report notebook with comments

- **requirements.txt** — Python dependencies
  - Use `pip install -r requirements.txt` for pip 
- **.gitignore** — Ignore data, logs, outputs  
- **README.md** — Project overview, setup instructions  
- **LICENSE** — MIT license
